There are millions of videos available on YouTube, which makes it difficult for users to keep track of the information they need from a particular video.

As a result, users may spend a lot of time re-watching videos to find the relevant information they need. This can be a frustrating experience and can waste valuable time. To solve this problem, we propose a web application that can create notes from YouTube videos.

The application will allow users to paste the URL of the YouTube video, and the application will generate notes from the video's transcript. The application will also provide the user with an option to download the notes as a text file.

This application will help users save time and effort by allowing them to quickly access important information from YouTube videos in the form of notes. This will make it easier for them to learn, study, and complete their projects efficiently.!
